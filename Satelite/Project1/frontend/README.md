# Frontend – Crawlability Checker (Project1)

Next.js 16 + React 19 + Tailwind 4. Consume la API del backend (puerto 8000).

## Instalación y ejecución

Ver **[README del proyecto](../README.md)** en `Satelite/Project1/` para requisitos, instalación del backend y variables de entorno.

**Resumen rápido:**

```bash
npm install
# Opcional: crear .env.local con NEXT_PUBLIC_API_URL=http://localhost:8000
npm run dev          # desarrollo (puerto 3000)
npm run build        # compilar
npm run start        # servir compilado
```

El backend debe estar corriendo en el puerto 8000 antes de usar la app.
